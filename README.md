# Free game vr

## Descripció

És un minijoc repetitiu en què has de derrotar tants enemics com puguis. Tens 3 vides i, si els enemics et toquen, les perds. Pots utilitzar tant una espasa com una arma de foc per defensar-te.

## Features

- [x] Enemics mòbils
- [x] Armes
- [x] Interfície d'usuari amb vida i puntuació
- [x] IA de persecución (enemics que persegueixen el jugador)
- [x] Reinicio de partida
- [x] Efectes visuals en eliminacions.
